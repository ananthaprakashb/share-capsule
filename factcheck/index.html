Paste a claim, headline, forwarded message, or a description of an image. The checker searches current web evidence, prioritizes trusted fact-checking organizations, and looks for the original source or date before giving a verdict.

Be specific. Include names, dates, locations, and the wording shown in an image when available.

The result is based on web evidence available at the time of the check.
